# Preferencia de Idioma: Español

Este skill establece que el usuario prefiere que toda la interacción y documentación técnica se realice en español.

## Reglas de Aplicación

1.  **Comunicación**: Todas las respuestas al usuario deben ser en español.
2.  **Planes de Implementación**: Los archivos `implementation_plan.md` deben redactarse íntegramente en español.
3.  **Mensajes de Commit**: Todos los mensajes de commit en Git deben escribirse en español (ej: "Mejora: implementar ISR..." en lugar de "Improvement: implement ISR...").
4.  **Documentación**: Walkthroughs, tareas y otros artefactos deben seguir esta misma regla.
